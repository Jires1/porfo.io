introduction

C++ is a general-purpose programming language that was developed by Bjarne Stroustrup in 1983. It is an extension of the C programming language and supports object-oriented programming, procedural programming, and generic programming. C++ is widely used in developing operating systems, system software, device drivers, embedded software, and video games.

hello world

Here’s a simple “Hello world” program in C++: 

#include 

int main() {
    std::cout << "Hello world!";
    return 0;
}
This program uses the cout object from the iostream library to output the string “Hello world!” to the console.

variables

In C++, variables are used to store data. A variable is defined by specifying its type and name. Here’s an example: 

int age = 25;
Here, type is the type of the variable (such as int, float, or double) and name is the name of the variable.

variable scope

The scope of a variable determines where it can be accessed in a program. In C++, variables can have either local or global scope. A local variable is declared inside a function or block and can only be accessed within that function or block. Here’s an example:

void myFunction() {
    int x = 5; // local variable
    std::cout << x;
}
A global variable is declared outside of any function or block and can be accessed from anywhere in the program. Here’s an example: 
int x = 5; // global variable
void myFunction() {
    std::cout << x;
}

constants

In C++, constants are used to define values that cannot be changed during program execution. The syntax for defining a constant is: 

const type name = value;
}
Here, type is the type of the constant (such as int, float, or double), name is the name of the constant, and value is its initial value.

data types

C++ supports several data types, including:

  • int: integer values (such as 1, 2, 3)
  • float: floating-point values (such as 3.14)
  • double: double-precision floating-point values (such as 3.14159)
  • char: single characters (such as ‘a’, ‘b’, ‘c’)
  • bool: boolean values (either true or false)

if…else statement

The if…else statement is used to execute code based on a condition. Here’s an example: 

if (x > y) {
    std::cout << "x is greater than y";
} else {
    std::cout << "y is greater than x";
}
This code compares the values of x and y. If x is greater than y, it outputs “x is greater than y”. Otherwise, it outputs “y is greater than x”.

while statement

The while statement is used to execute code repeatedly while a condition is true. Here’s an example: 

int i = 0;

while (i < 10) {
    std::cout << i << "\n";
    i++;
}
This code outputs the numbers from 0 to 9.

function declarations

Functions are used to group statements together into reusable blocks of code. Here’s an example:

int add(int x, int y) {
    return x + y;
}
This code defines a function called add that takes two integer parameters (x and y) and returns their sum.
